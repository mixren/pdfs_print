				File Merger
					by Mihails Zuravlovs

Программа "File Merger" создана для слияния/объединения множества файлов из разных папок в один файл.

Типы обЪединения:
	1) Лицевые страницы множества чертежей в формате ".pdf" в один PDF. Чертежи храняться в папке "./Rasejumi Ceham/"
	2) Файлы Запись Контроля Качества в формате ".xlsx" в один Excel документ со множеством вкладок. Храняться в папке "./DWR/"

Что объединять:
	1) Все найденные файлы одного типа из перечня выше.
	   В этом случае будет осуществлён поиск всех этих файлов в выбранной корневой и дочерних папках.
	2) Отдельно выбранные файлы.
	   В этом случае названия частей труб, типа "CWT.MP21-79-34.01.00", должны быть выписанны в первый столбик Excel документа.
	   Этот Excel документ будет использован для поиска нужных файлов.

Преимущества:
	Традиционно при необходимости печати множества чертежей или DWR документов нужно заходить в отдельные папки,
	открывать документы и печатать их по отдельности.
	С программой достаточно создать один Excel документ с названиями частей труб в столбик, запустить программу, подождать
	и распечатать все страницы из одного объединённого документа. Чтобы не писать названия частей труб в ручную,
	удобнее их скопировать из таблиц в папках сервера "/Стыки по спулам/" или подобных.

Результат слияния создаётся в коренной папке программы.
